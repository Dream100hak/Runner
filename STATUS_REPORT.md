## ? Special Notes

### Design Decisions
1. **Transform vs Physics**: Used Transform.position updates via Rigidbody for consistent, predictable movement
2. **Tap vs Hold**: 0.2s threshold chosen for natural mobile feel
3. **Stamina Drain**: 30/second allows ~3 seconds of full-speed dash
4. **Camera Offset**: Auto-calculated to preserve scene editor setup
5. **Collision Trigger**: Used OnTriggerEnter for responsive feedback

### Performance Optimizations Applied
- Raycasts use LayerMask for efficiency
- Component references cached in Awake
- Animator state checks minimize parameter updates
- Physics drag replaces constant force application
- No object instantiation in Update/LateUpdate
